# IMPORTANT: Run setup_environment.bat FIRST to install dependencies.

This kit provides a simplified, automated environment for merging Eressea report files (.cr) using the Magellan 2.1.1 ConsoleMerger.

## Contents
- **setup_environment.bat**: A setup wizard to prepare your folder structure.
- **run_auto_merge.bat**: The launcher for the merge tool.
- **auto_merge.py**: The logic script that processes files.

## Usage

1. **Run Setup**: Double-click `setup_environment.bat`.
2. **Add Data**: Copy your `etc` and `rules` folders from Magellan to the newly created folders.
3. **Run Merger**: Place `.cr` files in `input` and run `run_auto_merge.bat`.
